*ルール(知っている人は読まなくて良いゾ！)
	画面上を反射しながら移動するボールを、画面下部に落ちないように、ラケット（バー）を左右に操作して打ち返し、
	画面に並べられたブロックを破壊していくゲーム。
	すべてのブロックを破壊するとゲームクリア。


*操作方法など(読まなくても楽しめるゾ!)
	・まずはSTARTボタンでゲームスタート。
　	・十字キー左右でラケットが左右に動く。
	・ボールが画面下部に落ちてしまった時はSTARTボタンでリスタートができる。
	



	・実はゲーム中にSELECTボタンを押すと一瞬でゲームクリアとなるズル機能を搭載。良い子は押しちゃダメだぞ！
